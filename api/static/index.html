Welcome to the openyieldtables API


This is the API for the openyieldtables project.

The documentation of the project can be found here. The API documentation can be found here.


The following endpoints are available:

1. GET: https://yieldtables.org/v1/yield-tables-meta
2. GET: https://yieldtables.org/v1/yield-tables-meta/:id
3. GET: https://yieldtables.org/v1/yield-tables/:id
